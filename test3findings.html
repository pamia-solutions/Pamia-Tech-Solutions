Executive Summary

This report analyzes the e-commerce company’s quarterly performance data. Insights include the highest performing sales channels, average customer trends per quarter, and revenue efficiency per customer. Visualizations reveal which channels drive the most revenue and convert customers efficiently. Based on these analyses, strategic recommendations prioritize investment in high-performing channels to maximize revenue and profitability. Specifically, the online channel generates the highest total revenue ($5,571,130), while Retail achieves the highest revenue per customer ($314.22), which highlights which channels are clearly more profitable overall and per customer.

2 Creating Professional Charts

a)

# Calculate totals first
channel_totals <- aggregate(revenue ~ channel, data = business_data, sum)
# Then create the chart
ggplot(channel_totals, aes(x = channel, y = revenue)) +
  geom_bar(stat = "identity") +
  labs(title = "Total Revenue by Channel", 
       x = "Sales Channel", y = "Revenue ($)")

Answer (a) : Based on the bar chart showing total revenue by channel, the Online channel generates the highest overall revenue. This makes it apparent that the company should in fact consider allocating more investment into the Online channel, because of the fact that it appears to be the strongest performer, clearly offering the greatest potential for continued profitability and overall growth.

3 Action Plan

a)

# Load knitr for clean tables
library(knitr)

# 1) Total revenue by channel
totals <- aggregate(revenue ~ channel, data = business_data, sum)

# 2) Average customers by channel
avg_customers <- aggregate(customers ~ channel, data = business_data, FUN = mean, na.rm = TRUE)

# 3) Average revenue per customer
business_data$revenue_per_customer <- business_data$revenue / business_data$customers
avg_rpc <- aggregate(revenue_per_customer ~ channel, data = business_data, FUN = mean, na.rm = TRUE)

# 4) Average conversion rate by channel
avg_conversion <- aggregate(conversion_rate ~ channel, data = business_data, FUN = mean, na.rm = TRUE)

# 5) Merge all metrics into a single table
summary_table <- merge(totals, avg_customers, by = "channel")
summary_table <- merge(summary_table, avg_rpc, by = "channel")
summary_table <- merge(summary_table, avg_conversion, by = "channel")

# 6) Rename columns for presentation
colnames(summary_table) <- c("Channel", "Total Revenue", "Average Customers", 
                             "Average Revenue per Customer", "Average Conversion Rate")

# 7) Display a professional table in R Markdown
kable(summary_table, digits = 2, caption = "Summary of Key Metrics by Channel")
